# Netrix Systems Limited Website

Marketing site for Netrix Systems Limited, built with Vite, React, TypeScript, Tailwind CSS, and shadcn/ui.

## Getting started

```sh
npm install
npm run dev
```

The development server runs on `http://localhost:8080` by default.

## Available scripts

```sh
npm run dev
npm run build
npm run preview
npm run lint
npm run test
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Vitest
- Playwright
